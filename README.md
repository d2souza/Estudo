🕵️ Web Scraping & Automation Portfolio
Este repositório contém uma coleção de projetos de automação web desenvolvidos com Python e Selenium. 
O foco aqui é a extração de dados em sites dinâmicos, simulando o comportamento humano para superar barreiras de renderização em JavaScript.

🛠️ Tecnologias e Conceitos Dominados
Engine: Selenium WebDriver (Chrome/Edge).

Estratégias de Seleção: XPath avançado, CSS Selectors e IDs dinâmicos.

Performance: Implementação de WebDriverWait e Expected Conditions.

Navegação: Manipulação de frames, janelas, alertas e scroll infinito.

Modo Headless: Configuração de scripts para rodar em segundo plano, otimizando recursos.

🚀 Meus Projetos
🛒 1. Monitor de Ofertas - Mercado Livre
Automação que realiza buscas personalizadas, extrai uma lista estruturada de produtos.

Desafio Superado: Lidar com a estrutura de classes dinâmicas do Mercado Livre e realizar a paginação automática para coletar múltiplos resultados.

Saída: Arquivo .csv com Nome, Preço, Link.

📰 2. News Scraper Diário
Um agregador que acessa portais de notícias e extrai as manchetes em destaque no exato momento da execução.

Desafio Superado: Capturar elementos que só são carregados após o scroll da página (Lazy Loading).

Destaque: Uso de Headless Mode para uma execução rápida e silenciosa.

🏨 3. Penedo Booking Finder (WIP)
Ferramenta para automação de busca de reservas em pousadas na região de Penedo/RJ.

Status: 🛠️ Em Desenvolvimento.

Desafio Atual: Interação com calendários complexos e date-pickers que utilizam máscaras de input.

O que estou aprendendo: Manipulação de execuções de JavaScript via Selenium (execute_script) para forçar seleções de datas.

📂 Organização do Repositório
drivers/: Pasta para organização dos WebDrivers (ou uso de webdriver-manager).

logs/: Registro de erros de execução para facilitar o debug.

requirements.txt: Lista de dependências para fácil replicação do ambiente.
